# om30-saudesimples

Userscripts para uso em sistema interno.

## 📋 Requisitos

É necessário ter o **Tampermonkey** instalado no navegador:

* Chrome: https://chromewebstore.google.com/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo
* Firefox: https://addons.mozilla.org/pt-BR/firefox/addon/tampermonkey/

## 💡 Observações

* Certifique-se de que o Tampermonkey está ativo no navegador e que os scripts de usuário estão sendo permitidos.
* Após instalar, recarregue a página do sistema interno para aplicar os scripts.
