# om30-saudesimples

> Userscripts para otimizar o uso do sistema interno Saúde Simples.

---

## 📋 Pré-requisitos

Você precisa ter o **Tampermonkey** instalado no seu navegador:

| Navegador | Link |
|-----------|------|
| Chrome | [Instalar via Chrome Web Store](https://chromewebstore.google.com/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo) |
| Firefox | [Instalar via Firefox Add-ons](https://addons.mozilla.org/pt-BR/firefox/addon/tampermonkey/) |

---

## 🚀 Scripts disponíveis

Clique no nome do script para instalar diretamente pelo Tampermonkey.

### 🔎 [Busca de Pacientes Agendados](https://github.com/otowm/om30-saudesimples/raw/refs/heads/main/Sa%C3%BAde%20Simples%20-%20Busca%20Pacientes%20Agendados.user.js)

Facilita a localização de pacientes com agendamentos no sistema.

---

### 📊 [Relatórios e Cores de Área](https://github.com/otowm/om30-saudesimples/raw/refs/heads/main/Sa%C3%BAde%20Simples%20-%20Relat%C3%B3rios%20e%20Cores%20de%20%C3%A1rea.user.js)

Melhora a visualização de relatórios com identificação por cores de área.

---

### 🆎 [Atualizador de Título + Logo](https://github.com/otowm/om30-saudesimples/raw/refs/heads/main/Sa%C3%BAde%20Simples%20-%20Atualizar%20t%C3%ADtulo%20+%20logo.user.js)

Atualiza o título da aba e o logo exibido no sistema.

---

## ⚙️ Como instalar

1. Instale o Tampermonkey no seu navegador (links acima)
2. Clique no link do script desejado — o Tampermonkey abrirá automaticamente a tela de instalação
3. Confirme clicando em **Instalar**
4. Recarregue a página do sistema interno

> ⚠️ Certifique-se de que o Tampermonkey está **ativo** e que os userscripts estão **permitidos** nas configurações do navegador.
